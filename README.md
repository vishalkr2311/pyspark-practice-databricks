# PySpark Practice in Databricks

## Overview

This repository contains my hands-on PySpark and Spark SQL practice completed using Databricks.

The objective of this repository is to strengthen my understanding of DataFrame APIs, Spark SQL, and common data engineering transformations that are frequently used in real-world Azure Databricks projects.
